# 07. Node · Express

JavaScript로 서버를 만드는 런타임(Node.js)과 웹 프레임워크(Express).

## 배우는 것
- Node.js 실행 환경, npm 패키지 관리
- Express 서버 띄우기, 라우팅(GET/POST)
- 미들웨어, 요청/응답 처리
- 프론트-서버 연동, REST API

## 코드
- `index_04~09.js` : 단계별 서버 예제
- `01/`, `PR/` : 실습 폴더
