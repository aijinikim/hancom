# 19. Streamlit

파이썬 코드만으로 **데이터 웹앱**을 빠르게 만드는 프레임워크. HTML/JS 없이 대시보드·시각화 UI 제작.

## 배우는 것
- `streamlit run`으로 앱 실행, 위젯(입력/버튼/슬라이더)
- 텍스트·데이터프레임·차트 출력
- 그래프 시각화(라인/바 등)
- 실시간 상호작용 UI

## 코드
- `v19_01_streamlit_b.py` : Streamlit 기본(위젯·출력)
- `v19_02_streamlit_graph.py` : 그래프·차트 시각화

## 실행
```
streamlit run v19_01_streamlit_b.py
```
