# 14. YOLO

실시간 객체 탐지(Object Detection) 모델. 이미지·영상에서 사물을 찾아 박스로 표시한다.

## 배우는 것
- YOLO 기초: 사전학습 모델로 탐지, 결과 시각화
- YOLO 심화: SAHI(슬라이스 추론)·FastSAM 등 고급 기법
- 커스텀 데이터 학습 흐름

## 코드
- `basic/` : YOLO 기초 스크립트
- `advanced/` : 심화 구현
- `yolo11n.pt` · `yolo11s.pt` : 사전학습 가중치
