# 20. Gradio

**ML 데모 UI**를 파이썬 함수 하나로 만드는 라이브러리. 입력→함수→출력 인터페이스를 자동 생성.

## 배우는 것
- `gr.Interface` / `gr.Blocks`로 UI 구성
- 입력·출력 컴포넌트(텍스트/이미지/파일)
- 함수 연결 → 즉시 웹 데모
- 이미지 처리·모델 추론 데모, 공유 링크(share)

## 코드
- `v20_01_gradio.py` ~ `v20_04_gradio.py` : 단계별 데모(기본 → 이미지 처리 등)

## 실행
```
python v20_01_gradio.py
```
