# 13. Data

데이터 다루기와 이미지 전처리·증강(augmentation). 모델 학습용 데이터를 준비한다.

## 배우는 것
- 이미지 로드/저장, 로컬 데이터 수집
- 밝기 조정·회전·좌우반전 등 augmentation
- 라벨/클래스 관리(`classes.txt`)

## 코드
- `data_aug.py` : 이미지 증강
- `get_local_data.py` : 로컬 데이터 수집
- `img_brightness/flip/rotated` : 결과 샘플
- `captured_images/` : 캡처 이미지
